# 📖 Study Guide


## 🧾 Step-by-Step

- Log into GitHub
- Create a new repository
- Define name and description
- Choose visibility
- Add README
- Make first commit
- Organize structure


## 📘 Best Practices

- Use clear naming
- Write a structured README
- Keep documentation concise
- Maintain clean commits


## 📗 Glossary

| Term       | Definition         |
| ---------- | ------------------ |
| Repository | Project storage    |
| Commit     | Saved change       |
| Branch     | Parallel version   |
| README     | Main documentation |
